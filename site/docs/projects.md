## Основные проекты

### 1. MyBox

**Статус:** Базовая платформа
**Описание:** Персональный сервер на базе Mac Mini / Mac Studio для развёртывания приватных приложений и AI-решений.

**Ключевые люди:**
- **Евгений Ерин** (@EugeneErin) — продакт-менеджер/координатор проекта
- **Вова Белов** (@vlbelov) — технический лид, разработка платформы
- **Юрий Максимов** — владелец продукта, основной стейкхолдер

**Ключевые характеристики:**
- Аппаратная платформа: Mac Mini (16 GPU, 16 ГБ ОЗУ) или Mac Studio (до 80 GPU, 512 ГБ ОЗУ)
- Суверенное решение — все данные хранятся локально
- Мобильное приложение для управления установкой сторонних приложений

**Связь с другими проектами:** Является инфраструктурной основой для всех остальных проектов (Personal AI, Бэкапы, FAM, Media Room).

### 2. Personal AI / Майя

**Статус:** Основной проект чата
**Описание:** Персональный AI-ассистент, который ориентируется в личных данных пользователя (переписки, записи встреч, голосовые сообщения).

**Ключевые люди:**
- **Наиль** (@nfatkullov) — инициатор проекта, продуктовое видение
- **Миша Рейнганд** (@reyingand) — основной разработчик, архитектор решения
- **Юрий Максимов** — основной пользователь-тестировщик, для которого делается ассистент
- **Константин Степаненко** (@kas_stepanenko) — DevOps/инфраструктура, тестирование (?)
- **Антон** (@zemerov1) — AI-эксперт, консультации по LLM (автор @DoubleMaksimovBot)
- **Артём Южаков** — разработчик (присоединился позже)
- **Данил Журавлев** — подрядчик по разработке (?)

**Цели проекта:**
- Развернуть LLM локально на MyBox
- Научить ассистента отвечать на вопросы по личным данным ("когда я обсуждал проект X")
- Создать "безопасного Джарвиса" для работы с приватной информацией

**Технологический стек:**
- LLM: QWEN, Mistral 7B и другие модели
- Транскрибация: кастомизированный Whisper
- Векторная база данных для хранения контекста

**Этапы развития:**
1. Транскрибация + саммари аудио
2. Интеграция с бэкапами мессенджеров
3. Распознавание спикеров по голосу
4. Полноценная база знаний

→ См. [Эволюция концепции](./vision-evolution) для детальной хронологии

### 3. Бэкапы / Gravitonum

**Статус:** Действующий проект
**Описание:** Система резервного копирования данных из мессенджеров (Telegram, WhatsApp) с сохранением структуры и метаданных.

**Ключевые люди:**
- **@GravitonumPlatform** — бот/аккаунт платформы
- **Наиль** (@nfatkullov) — упоминает как связанный проект
- **Евгений Ерин** (@EugeneErin) — интеграция с MyBox

**Функциональность:**
- Выгрузка данных из Telegram (текст, голосовые, медиа)
- Парсинг и нормализация данных в связную структуру
- Хранение в структурированном формате для дальнейшего анализа

**Интеграция с Personal AI:**
- Данные бэкапов — основной источник для обучения и работы AI-ассистента
- Голосовые сообщения транскрибируются и добавляются в базу знаний

### 4. Media Room

**Статус:** Планируемая ветка
**Описание:** Система записи и хранения встреч на базе MyBox (аналог PLAUD, но локальный).

**Ключевые люди:**
- **Наиль** (@nfatkullov) — автор концепции
- **Константин Степаненко** (@kas_stepanenko) — подбор микрофонов, тестирование
- **Юлия Безус** (@ubezus) — активный пользователь PLAUD, интервью по UX

**Концепция:**
- MacMini + микрофон для записи встреч
- Автоматическая транскрибация и саммаризация
- Хранение записей в структурированном архиве
- Интеграция с Personal AI для поиска по записям

**Референсы:** PLAUD, Bee, Limitless (американские аналоги)

---

### 5. FAM (Financial Assets Management)

**Статус:** Приложение MyBox
**Описание:** Приложение для учёта и управления финансовыми активами.

**Ключевые люди:**
- **Вова Белов** (@vlbelov) — упоминает чат "FAM запуск на MyBox"
- Разработчики FAM не идентифицированы в данном чате

**Интеграция с Personal AI:**
- Данные FAM могут обогащать базу знаний ассистента
- Голосовой интерфейс: "Майя, попроси FAM сообщить текущий баланс активов"

---

### 6. Транскрибация (Daisy)

**Статус:** Реализован
**Описание:** Сервис преобразования аудио в текст на базе кастомизированного Whisper.

**Ключевые люди:**
- **Евгений Ерин** (@EugeneErin) — координация с AI-командой
- **Юрий Максимов** — предложил название "Даша" (Daisy)
- **Миша Рейнганд** (@reyingand) — интеграция в пайплайн Personal AI
- AI-юнит команды (не идентифицирован поимённо) — разработка кастомного Whisper

**Характеристики:**
- Улучшенное распознавание русского языка
- Telegram-бот для тестирования: @daisytranscribe_bot
- Работает на коммерческих проектах

**Планируемые улучшения:**
- Диаризация (разделение на спикеров)
- Распознавание личностей говорящих (требует 2-3 месяца на обучение модели)

---

### 7. SmartPlatform

**Статус:** Проприетарный продукт
**Описание:** Внутренний продукт AI-команды, который может использоваться для построения базы знаний и масштабирования LLM-решений.

**Ключевые люди:**
- AI-команда (не идентифицирована поимённо в чате)

## Резюме

Все проекты объединены общей концепцией **суверенного персонального AI** — данные пользователя остаются на его устройстве (MyBox), а AI-ассистент "Майя" помогает ориентироваться в этих данных через голосовой и текстовый интерфейс.
