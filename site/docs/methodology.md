## Фаза 1: Подготовка данных

Цель: получить полную текстовую историю всех сообщений чата.

### 1.1 Конвертация HTML в текст

Telegram-экспорт (HTML-файлы) преобразован в читаемый текстовый формат с сохранением всей метаинформации.

**Стек:** Python 3.12, BeautifulSoup4

### 1.2 Транскрибация голосовых сообщений

- Голосовые сообщения → текст
- Очистка транскрипции
- Инъекция в историю чата (на место оригинального голосового)

**Стек:** Whisper / speech-to-text API

### 1.3 Обработка видео

- Извлечение аудиодорожки из видео
- Транскрибация аудио в текст
- Очистка транскрипции

**Стек:** FFmpeg (извлечение аудио), Whisper (транскрипция)

### 1.4 Обработка файлов

- Преобразование документов в Markdown
- Ручной разбор релевантности (нужно / не нужно для анализа)
- Включение релевантных файлов в контекст

**Стек:** Pandoc / ручная конвертация

### Результат фазы 1

Полная текстовая история сообщений чата со всеми вложениями.

## Фаза 2: Анализ содержания

Цель: извлечь структурированную информацию из истории чата.

### 2.1 Анализ участников

- Выделение всех участников чата
- Определение главных и второстепенных участников
- Составление сводки по каждому участнику

→ Результат: [Участники чата](./chat-participants)

### 2.2 Выделение проектов

- Идентификация всех проектов, обсуждавшихся в чате
- Описание каждого проекта

→ Результат: [Проекты](./projects)

### 2.3 Фазы основного проекта

- Определение ключевых этапов развития основного проекта
- Хронология событий и решений

→ Результат: [Эволюция концепции](./vision-evolution)

### 2.4 Анализ позиций участников

- Разбор дискуссий по веткам
- Выявление ожиданий каждого участника от проекта
- Фиксация: кто что думал о проекте в какой момент
- Анализ различий в видении проекта

→ Результаты:
- [Позиции участников](./positions-by-participant)
- [Матрица позиций](./positions-matrix)
- [Хронология позиций](./positions-evolution)
