# Инструкции для проекта

## Структура проекта

### Директория utils/

Все новые вспомогательные скрипты (утилиты, хелперы, вспомогательные функции) должны создаваться в директории `utils/`.

Эта папка предназначена для хранения всех utility скриптов проекта.

**Правило:** При создании нового вспомогательного скрипта, ВСЕГДА размещайте его в `utils/`.
