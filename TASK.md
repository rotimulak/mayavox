Постановка задачи


Контекст

Есть Telegram-чат проекта "Майя" (~17 участников), в котором на протяжении нескольких месяцев обсуждалось создание персонального AI-ассистента. В чате накопилось большое количество сообщений: текстовые, голосовые, видео, документы.

Проблема: информация разрознена, сложно восстановить как менялось видение продукта, какие позиции занимали участники в разные моменты времени, кто за что отвечает, какие проекты обсуждались и как они связаны.


Цель

Создать базу знаний по проекту, которая позволит быстро найти нужную информацию, понять историю принятия решений, увидеть эволюцию концепции и отвечать на вопросы по проекту через бота.


Задачи

1. Подготовка данных — получить полную текстовую историю чата
2. Анализ содержания — создать набор структурированных документов
3. Доступ к результатам — опубликовать сайт и запустить Telegram-бота


Ограничения

Данные конфиденциальны — обработка локально. Анализ должен быть объективным, без интерпретаций. Бот отвечает только на основе фактов из анализа.


Критерии успеха

Любой участник может получить ответ на вопрос о проекте в боте.
