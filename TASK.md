# Постановка задачи

## Контекст

Есть Telegram-чат проекта "Майя" (~17 участников), в котором на протяжении нескольких месяцев обсуждалось создание персонального AI-ассистента. В чате накопилось большое количество сообщений: текстовые, голосовые, видео, документы.

**Проблема:** информация разрознена, сложно восстановить:
- Кто за что отвечает
- Какие проекты обсуждались и как они связаны
- Как менялось видение продукта
- Какие позиции занимали участники в разные моменты времени

## Цель

Создать структурированную базу знаний по проекту, которая позволит:
1. Быстро найти нужную информацию
2. Понять историю принятия решений
3. Увидеть эволюцию концепции
4. Отвечать на вопросы по проекту через AI-бота

## Задачи

### 1. Подготовка данных

- [ ] Экспортировать чат из Telegram (HTML-формат)
- [ ] Конвертировать HTML в читаемый текст с сохранением метаданных
- [ ] Транскрибировать голосовые сообщения
- [ ] Извлечь и транскрибировать аудио из видео
- [ ] Конвертировать приложенные документы в текст

**Результат:** полная текстовая история чата.

### 2. Анализ содержания

- [ ] Выделить и описать всех участников чата
- [ ] Определить роли: заказчики, координаторы, разработчики
- [ ] Идентифицировать все обсуждаемые проекты
- [ ] Описать связи между проектами
- [ ] Зафиксировать эволюцию концепции основного проекта
- [ ] Проанализировать позиции каждого участника
- [ ] Построить матрицу позиций по ключевым вопросам
- [ ] Отследить изменение позиций во времени

**Результат:** набор структурированных документов.

### 3. Доступ к результатам

- [ ] Опубликовать документацию на сайте
- [ ] Создать Telegram-бота для вопросов по анализу
- [ ] Настроить бота на работу только с фактами из анализа

**Результат:** удобный доступ к базе знаний.

## Ограничения

- Данные конфиденциальны — обработка локально
- Анализ должен быть объективным, без интерпретаций
- Бот отвечает только на основе фактов из анализа

## Критерии успеха

1. Любой участник может получить ответ на вопрос о проекте за минуту
2. История решений прослеживается
3. Позиции участников зафиксированы и верифицируемы
